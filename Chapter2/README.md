# 제 2고지

## 11단계. 가변 길이 인수 (순전파 편)
- 함수의 입력과 출력은 2개 이상일 수 있음
- 변수들을 리스트 혹은 튜플에 넣어 처리
- Add 클래스를 사용하기 위해 입력 변수를 리스트에 담거나 튜플을 반환받는 것은 자연스럽지 않음

## 12단계. 가변 길이 인수 (개선 편)
- Add 및 함수 클래스 사용자를 위한 개선
    - 기존: 인수를 리스트에 모아서 받고 결과를 튜플로 반환
    - 개선: 인수와 결과를 직접 주고받음
- 구현자를 위한 개선
    - 리스트를 unpack하여 `self.forward`호출
